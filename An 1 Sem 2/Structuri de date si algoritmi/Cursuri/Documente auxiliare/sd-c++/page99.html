InsertAfter and InsertBefore

The functions InsertAfter and InsertBefore both take two arguments. The first is a pointer to an element of the linked list and the second is a reference to the item to be inserted into the list. The item is inserted either in front of or immediately following the indicated list element. Program  gives the code for the InsertAfter and InsertBefore member functions.
