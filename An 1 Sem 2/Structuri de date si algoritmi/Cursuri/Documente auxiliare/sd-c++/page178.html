Member Variables

Objects of the ListAsLinkedList class contain one member variable, linkedList, which is a linked list of pointers to Objects. The linkedList is used to hold the items in the ordered list. Since a linked list is used, there is no notion of an inherent limit on the number of items which can be placed in the ordered list. Items can be inserted until the available memory is exhausted.
