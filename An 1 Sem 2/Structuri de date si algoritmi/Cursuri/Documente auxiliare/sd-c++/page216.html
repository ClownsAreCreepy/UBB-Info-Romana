Integral Keys

Out of all the C++ built-in data types, the so-called integral types  are the simplest to hash into integers. The integral data types are char and wchart, int of all sizes, and enumerations. Since the underlying representation of such data types can be viewed as an integer, the hash function f is trivial. The only wrinkle involves dealing with negative values.
