Sorter Class Hierarchy

This chapter describes nine different sorting algorithms. These are organized into the following five categories:

  • insertion sorts 
  • exchange sorts 
  • selection sorts 
  • merge sorts 
  • distribution sorts .
As shown in Figure , the sorter classes have been arranged in a class hierarchy that reflects this classification scheme.

  
Figure: Sorter Class Hierarchy

Program  defines five abstract sorter class templates: InsertionSorter<T>, ExchangeSorter<T>, SelectionSorter<T>, MergeSorter<T> and DistributionSorter<T>. Notice that these classes are all trivial (i.e., empty) extensions of the Sorter<T> abstract base class.
