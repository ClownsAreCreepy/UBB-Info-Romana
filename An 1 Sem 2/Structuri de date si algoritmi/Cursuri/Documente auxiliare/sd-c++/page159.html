Array Implementation

Program  defines an array implementation of a deque. The DequeAsArray class is defined using multiple inheritance: It is derived from both the Deque abstract base class and the QueueAsArray array-based implementation of a queue. The Deque class provides the interface; the QueueAsArray, the implementation.

  
Program: DequeAsArray Class Definition

The QueueAsArray implementation provides almost all of the required functionality. For example, Program  shows that the Head, EnqueueTail and DequeueHead operations of the DequeAsArray class are implemented by calling the appropriate QueueAsArray class operations.
