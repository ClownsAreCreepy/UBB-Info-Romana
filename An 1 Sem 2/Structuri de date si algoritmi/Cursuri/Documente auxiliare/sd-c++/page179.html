Inserting and Accessing Items in a List

Program  gives the implementation of the Insert member function of the ListAsLinkedList class. This function takes a reference to an Object which is to be added to the ordered list. As in the case of the ArrayAsLinkedList class, the object is added at the end of the ordered list. This is done simply by calling the Append function from the LinkedList<T> class.

The running time of the Insert function is determined by that of Append. In Chapter  this was shown to be O(1). The only other work done by the Insert function is to add one to the count variable. Consequently, the total running time for Insert is O(1).

In this case, the position is specified by a non-negative, integer-valued subscript expression.
