Linked List Implementation

 

This section presents a pointer-based implementation of ordered lists. Program  declares the ListAsLinkedList class. Like its array-based counterpart ListAsArray, the ListAsLinkedList class is a concrete class derived from the abstract base class List. The sets of member functions comprising the public interfaces of the ListAsArray and ListAsLinkedList classes are identical.
